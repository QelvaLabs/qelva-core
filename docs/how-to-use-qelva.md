# How to Use Qelva — Model App v1.0.1

This guide explains how to create an account, manage personas, track activity, and navigate Qelva’s privacy-native ecosystem.

---

## 1. Creating an Account
1. Open Qelva and select **Create Account**  
2. Enter a username (lowercase)  
3. Set a 6-digit PIN  
4. Confirm PIN  
5. Qelva generates an encrypted mnemonic locally  
6. Save your 12-word recovery phrase offline  
7. Continue into the app  

The recovery phrase is the only method to restore the account.

---

## 2. Logging In
1. Select **Sign In**  
2. Enter username  
3. Enter PIN  
4. Authenticate  

Forgot PIN? Restore using the recovery phrase.

---

## 3. Dashboard Overview
The dashboard shows:

- Total balance (across personas)  
- Number of personas  
- Privacy Score  
- Private transaction count  
- Quick actions  
- Recent activity  

---

## 4. Managing Personas
### Viewing Personas
Navigate to **Personas** to view all identities.

### Creating a Persona
1. Select **New Persona**  
2. Enter persona name  
3. Confirm with PIN  
4. Qelva generates a unique Solana address  

### Persona Details
- Copy wallet address  
- Refresh on-chain balance  
- Enable High Privacy mode  
- Add notes  
- Export private key (requires PIN)  
- Delete persona  

---

## 5. Activity History
View and filter transactions by:

- Status (All / Completed / Pending / Failed)  
- Type (Send / Receive / Mix)  
- Persona  

Export history via CSV.

---

## 6. Notifications
Notifications include:

- Transfer events  
- Persona creation  
- Security alerts  
- System updates  

Users can mark all as read or clear all.

---

## 7. Settings
Features include:

- Network selection (Mainnet / Devnet)  
- View recovery phrase  
- Reset all data  
- Request a Devnet airdrop
